# Sightmap

A YAML format that teaches agents how to see and use a web app's views, components, and API requests.

## Projects

- **[sightmap/spec](https://github.com/sightmap/spec)** — the spec, schema, examples, and conformance fixtures.
- **[sightmap/sightmap-js](https://github.com/sightmap/sightmap-js)** — JavaScript/TypeScript SDK + CLI ([`@sightmap/sightmap`](https://www.npmjs.com/package/@sightmap/sightmap) on npm).
- **[sightmap/sightmap-python](https://github.com/sightmap/sightmap-python)** — Python SDK + CLI (in development).
