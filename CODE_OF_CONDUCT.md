# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Scope

The Code of Conduct applies in all project spaces — this repository, GitHub Discussions, issues, pull requests, and any public or private channel where someone is representing the project.

## Reporting

If you experience or witness behavior that violates the Code of Conduct, please report it to the maintainers at **subtext@fullstory.com**. Reports are handled confidentially by the maintainers listed in [`MAINTAINERS.md`](MAINTAINERS.md).

Please include:

- What happened, with enough detail for the maintainers to understand the situation
- Where and when it happened (link to the issue, PR, or thread if applicable)
- Any other context you think is relevant

You do not need to be the target of the behavior to report it.

## Enforcement

Maintainers will follow the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) when determining consequences. Possible actions include a private reminder, a public warning, a temporary ban, or a permanent ban from project spaces.

Maintainers who do not enforce the Code of Conduct in good faith may be removed from the maintainer team.
